Morse Code Translator

This translator works in both directions. You can enter letters on the left or Morse Code on the right and the opposite box will be updated automatically.

Supported characters are letters A-Z in upper- and lower-case, numbers 0-9 and the punctuation marks .,?!

Delimit Morse characters with a single space, and Morse words with two spaces (which will be automatically converted to four).

When entering a Morse Code character, the symbol [ ] will be displayed until a valid code has been input.
